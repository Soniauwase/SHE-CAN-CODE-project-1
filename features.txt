## Visual Studio Code has many powerful features that make coding easier and faster. Here are some of the most useful ones.

 1. Multi-Cursor Editing:Lets you place multiple cursors at once
    Edit many lines at the same time.

 Example: change the same word in many places instantly

2. Integrated Terminal:Run commands (like Git) directly inside VS Code
No need to open another terminal.

Useful for:-git add,git commit and git push

3. Extensions Marketplace:Add extra features to VS Code.

Examples:-Python support,Live Server and Git tools

4. Git Integration:Built-in support for Git.

You can:See changes,Stage files,Commit code and Push to GitHub

No need to leave VS Code.

5. IntelliSense (Auto-completion):Suggests code as you type and
Helps avoid mistakes

Example:Suggests HTML tags, functions, variables.

 6. File Explorer
Easily manage your project files:Create, rename, delete files quickly

 7.Split Screen Editing:Open multiple files side by side
Compare or edit them at the same time
 
 8. Markdown Preview:View your README.md nicely formatted
Shortcut: Ctrl + Shift + V
